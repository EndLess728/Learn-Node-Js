# Node.js Learning Path (From React Native)

## 1) Runtime and Basics

- Event loop and non-blocking I/O
- CommonJS vs ESM
- Modules and package management

## 2) HTTP and Express

- Build routes, controllers, and services
- Request validation basics
- Error handling strategy

## 3) Data Layer

- Learn SQL (PostgreSQL) or NoSQL (MongoDB)
- Add a repository/data-access layer
- Handle migrations and seed data

## 4) Auth and Security

- JWT auth flow
- Password hashing with bcrypt
- Rate limiting, CORS, and headers

## 5) Testing

- Unit tests for services
- Integration tests for API routes
- Add test scripts and CI checks

## 6) Production Readiness

- Logging, monitoring, and health checks
- Environment-based config
- Docker and deployment basics
